HTTP Requests

When calling UDFs via HTTP requests, it's possible to specify geometries to do spatial queries and return data for specified areas.

Specify geometry‚Äã

There's 2 ways to specify geometries. The first method involves specifying XYZ tiles as path parameters and the second method entails passing a geometry query parameter.
